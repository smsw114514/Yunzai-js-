# 自用Yunzai-js
以plugins\example目录为主的js插件
以ai编写

# onlyAtBot.js
## 介绍 
当用户@bot时，bot引用消息并回复“干什么！！！”

## 使用
- 在Yunzai根目录运行
- 运行完后记得重启
```shell
curl.exe -L -o "plugins/example/onlyAtBot.js" https://raw.githubusercontent.com/smsw114514/Yunzai-js-/7a8af36ffcf2bae79f4880dd250c4a9793cf9a01/onlyAtBot.js
```
